📦 Eber-Ride
Eber-Ride is a full-stack ride-sharing application built with Angular and Node.js.. It allows users to book rides, track drivers in real-time, and manage their profiles seamlessly.


🚀 Features
User registration and login

Real-time ride tracking via sockets

Stripe integration for payments

Admin dashboard for ride management

Responsive UI for mobile and desktop

🛠️ Tech Stack
Frontend	Backend	Database Payments
Angular	Node.js	MongoDB	Stripe

⚙️ Setup Instructions
bash
# Clone the repo
git clone https://github.com/shreya29k/Eber-ride.git
cd Eber-ride

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Start backend
npm run dev

# Start frontend
ng serve
